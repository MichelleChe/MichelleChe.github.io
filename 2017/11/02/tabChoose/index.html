发表于 2017-11-02 | 分类于 js

这个选项卡比较简单，里面的代码逻辑也是很简单的，新手阅读也是能够读懂的。我个人觉得插件就像是你用vue等框架写的组件，目的就是为了能够多次调用，而不用去写那些多余的冗余的代码，还有一点很重的就是传参，通过参数来控制我想要的一切就是插件或者组件的重点所在，所以在一开始的时候就你需要用到的参数定好，作为构造函数的属性，然后在原型中定义他的方法，这个设计模式是最常用的一种方式也是很好的一种方式。还有一点就是最为新手你写好了代码最好找个厉害的人给你看看你的代码，规范你的代码，这对你很有帮助的。

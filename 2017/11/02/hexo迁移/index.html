hexo换了一台电脑怎么办

发表于 2017-11-02 | 分类于 hexo

因为公司的电脑不好用，重装系统后很多软件装不上，有些环境也是无法运行，于是自己买了台Mac。这里就介绍一下换了一台电脑那我的hexo搭建的博客怎么办呢，其实方法很简单的。我去网上看了很多的帖子，也是看的我一头雾水，所以最后我采取了简单粗暴的办法。

1、前提

这里的方法的前提是你之前的文件你还能找到，不然这个方法不适合你哈！

2、按照网上的方法一步一步的搭建好你的博客

这里仅仅就是安装好你的hexo，能够打开看见最原始的页面即可

3、复制文件

将你之前的_config.yml、scaffolds、source、themes这四个文件复制过来替换掉，然后重启一下你的hexo服务 hexo s即可，轻松搞定

如果打开出现

就是你的默认的IP被占用了，只需要打开的时候换个IP即可hexo s -p 5000

在本地一切都好当我准备发布这篇文章的时候有遇见了新的问题，就是我的git报错

这时需要安装一个插件然后再次hexo g,hexo d就能够成功的发布你的文章。然后打开你的http:xxx.github.io就能够看见你刚刚编辑的文章已经发成功了

树形组件

发表于 2017-11-02 | 分类于 js

最近没什么事就天天练习js，最近在看面向对象的编程，就用面向对象的思想写了个树形组件效果图

代码地址https://git.coding.net/michelle06/treeComponent.git

HTML中使用的时候new一个tree，第一个参数是你建的树形的id第二个是参数你要建的树的值，一个数组

js中Tree是一个构造函数，构造函数和普通函数不同的地方在与他的首字母是大写的这样易于区分

然后用原型给你的Tree添加方法

open和close分别控制着树形父级前面是图标也控制着子节点是隐藏还是显示这样有一个好处是一个class同时控制避免更多的操作
